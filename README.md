# DestinoFinal Portfolio

A transparent, dependency-free portfolio demonstrating responsive frontend,
audio capture UX, wallet connection states and documented backend concepts.

## Run locally

Open `index.html` in a modern browser. Microphone recording requires a secure
context (`https://` or localhost).

## Validation

```bash
node --check app.js
```

All work shown is labeled as prototype work. No private keys, seed phrases or
production wallet signatures are used.
